# Video & Music Downloader Telegram Bot

A simple Telegram bot to download videos and audio (MP3) from YouTube, Facebook, Instagram, TikTok, and many other sites.

## Features
- Download videos in different qualities (Low, Medium, High)
- Convert YouTube videos to MP3 audio
- Supports many social media platforms
- Clean and easy to use

## How to Use
1. Start the bot with `/start`
2. Send any video link
3. Choose Video or Audio option
4. Wait for the download

## Local Setup

```bash
# Install requirements
pip install yt-dlp python-telegram-bot

# Run the bot
python bot.py
